What is Polymorphism? 

Polymorphism means "many forms". It allows objects of different classes to be treated as objects of a common super class, typically when they share a common interface or method name.
--------------------------------------------------------------------------------------------------------

Why Use Polymorphism? 

-> Makes code flexible, reusable, and maintainable
-> Allows different object types to be used interchangeably
-> Supports extensibility: add new types with minimal code changes
